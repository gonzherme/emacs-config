*Native Compilation*
=emacs-plus= comes by default with Native Compilation. If switch =emacs= version, make sure to download with native compilation

Directly translates Emacs Lisp (Elisp) code into machine code
+ massive performance boost (2x to 5x)
+ smoother text rendering
+ much faster package execution (LSP integration)
+ everything will feel snappier, reducing UI stutter
+ faster startup time
** OS Integration & Paths
#+begin_src emacs-lisp
;; macOS GUI applications do not naturally inherit environment variables (like $PATH)
;; from your terminal. This package forces Emacs to read your shell's profile
;; so tools like node, python, and go work as expected.
;; NOTE: This will add a slight delay to startup as it has to spin up an interactive shell.
(use-package exec-path-from-shell
  :config
  (exec-path-from-shell-initialize))


;; Tell Emacs where to look for Homebrew binaries and GNU utilities on macOS
(when (eq system-type 'darwin)
  ;; macOS defaults to older BSD utilities. If you installed GNU coreutils via Homebrew 
  ;; (for tools like a modern `ls` or `gls`), Emacs needs to know where they live.
  ;; We target the Apple Silicon specific Homebrew path.
  (let ((gnu-path "/opt/homebrew/opt/coreutils/libexec/gnubin"))    
    ;; Check if the directory actually exists so it doesn't throw errors on Intel Macs or Linux
    (when (file-directory-p gnu-path)      
      ;; 1. Update the internal terminal's $PATH so tools run inside Emacs find GNU utils first
      (setenv "PATH" (concat gnu-path ":" (getenv "PATH")))      
      ;; 2. Update Emacs's own command search path so it natively uses GNU utils over macOS defaults
      (add-to-list 'exec-path gnu-path))))
#+end_src

integration mac reminders
#+begin_src emacs-lisp
(define-derived-mode apple-reminders-mode org-mode "Reminders"
  "A specialized Org mode that automatically fetches Apple Reminders."
  (let* ((cli-path (or (executable-find "remindctl")
                       "/opt/homebrew/bin/remindctl"))
         (inhibit-read-only t))
    
    (unless (file-exists-p cli-path)
      (error "The remindctl CLI was not found."))
      
    ;; Fetch and insert the reminders
    (erase-buffer)
    ;; file formatting
    (insert "📅 Reminders Due Today\n")
    
    (let ((output (shell-command-to-string (concat cli-path " today"))))
      (if (string-blank-p output)
          (insert "No reminders due today! 🎉\n")
        (insert output)))
        
    (goto-char (point-min))
    
    ;; Trick Emacs into thinking the file is unmodified so it doesn't ask to save
    (set-buffer-modified-p nil))
    
  ;; Lock the file as read-only
  (read-only-mode 1))

;; Tell Emacs to strictly use this mode whenever a file named "reminders.org" is opened
(add-to-list 'auto-mode-alist '("/reminders\\.org\\'" . apple-reminders-mode))
#+end_src


** Startup Optimization (Garbage Collection)
Emacs has a notoriously low default Garbage Collection (GC) threshold (about 800KB). Because loading packages generates a lot of garbage, Emacs will pause hundreds of times during startup just to clean up memory.
#+begin_src emacs-lisp
  ;; Make startup faster by reducing the frequency of garbage collection
  (setq gc-cons-threshold (* 100 1024 1024)
        read-process-output-max (* 1024 1024))

  ;; Reset GC to a reasonable normal value after startup
  (add-hook 'emacs-startup-hook
            (lambda ()
              (setq gc-cons-threshold (* 2 1024 1024))))
#+end_src

*** Settings to override theme
Theme changes, for example =auto-dark=, override settings such as: font, cursor color, tab colors, etc
The function below is added as a hook to =auto-dark= to ensure they're not overridden on theme switches

#+begin_src emacs-lisp
(defun my/apply-theme-tweaks ()
  ;; force fonts
  (set-face-attribute 'default nil :font "JetBrains Mono" :height 160 :weight 'medium)
  (set-face-attribute 'variable-pitch nil :font "SF Pro Text" :height 180 :weight 'normal)
  (set-face-attribute 'bold nil :weight 'semi-bold)

  ;; force org code blocks to use JetBrains Mono, so they render in minibuffer searches
  (set-face-attribute 'org-block nil :font "JetBrains Mono" :inherit 'default)
  
  ;; remove fringes globally
  (set-face-attribute 'fringe nil :background 'unspecified)

  ;; Fetch the current theme's backgrounds dynamically
  (let ((bg-main (face-background 'default nil t))
        (bg-inactive (face-background 'mode-line-inactive nil t)))

    ;; Base Tab Bar
    (set-face-attribute 'tab-bar nil
                        :background bg-inactive
                        :foreground 'unspecified
                        :box 'unspecified
                        :font "JetBrains Mono"
                        :height 150)

    ;; Active Tab (Uses main background + your custom accent)
    (set-face-attribute 'tab-bar-tab nil
                        :background bg-main
                        :foreground "#E5A822"
                        :box 'unspecified
                        :overline "#E5A822"
                        :weight 'bold)

    ;; Inactive Tabs (Matches inactive background to blend in)
    (set-face-attribute 'tab-bar-tab-inactive nil
                        :background bg-inactive
                        :foreground 'unspecified
                        :box 'unspecified
                        :overline nil
                        :weight 'normal))  
)
#+end_src

My custom margins
#+begin_src emacs-lisp
;; Define how wide your code usually is (standard is 80 or 100)
(defvar my-text-width 100
  "The expected width of your text block for perfect visual centering.")

(defun my/update-window-margins (&optional _frame)
  "Dynamically center text by setting equal visual left and right margins."
  (walk-windows
   (lambda (win)
     (let* ((width (window-total-width win))
            ;; Subtract text width from window width, divide by 2 to center.
            ;; If the window gets too small, fallback to a minimum 1 column margin.
            (margin (max 1 (/ (- width my-text-width) 2))))
       (set-window-margins win margin margin)))
   nil 'visible))

;; 3. Hook it so it runs automatically
(add-hook 'window-configuration-change-hook #'my/update-window-margins)

;; 4. Remove fringes 
(fringe-mode 0)

;; 5. Run once at startup
(my/update-window-margins)
#+end_src

** LSP
*** LSP Concept
Language Server Protocol (LSP), introduced by Microsoft in 2016.Before, features like auto-complete, go-to-definition or syntax checking had to be written in plugins, different ones for different editors.

Instead of writing a unique plugin for every editor-language combination, a language community only needs to write one LSP server

*Client and Server*
- LSP Client: your text editor (VSCode, Emacs, ...)
- LSP Server: a separate background process that deeply understands the language. Developed directly by the developers of the programming language. For example: =pyright= for Python, =gopls= for Go, =tsserver= for TypeScript

*Real-time*
The text editor and LSP server processes are in constant communication. When you open a file like =main.py=, your editor talks to the LSP server process to get syntax highlighting and all the features. When you interact with the code, like hovering over a variable to see its type, your editor sends the LSP server process an event like "get type info of variable =x=", and LSP server process will analyze and answer.

*Different Language Features*
Different languages will have different LSP server implementations, so they will most likely have different features.

** Tree-sitter
*Concept*
- Historically, text editors relied RegEx to figure out how to highlight your code, indent it, or fold it

*Benefits*
- better highlighting
- better indentation
- better code folding

Making opening images in org mode doable with a left click
#+begin_src emacs-lisp
;; Make Org inline images single-left-clickable
(defun my-org-image-click-fix (&rest _args)
  "Add a keymap to Org image overlays to allow opening with a single left click."
  (let ((image-map (make-sparse-keymap)))
    ;; Bind left-click to move the cursor to the mouse and open the link
    (define-key image-map [mouse-1]
      (lambda (event)
        (interactive "e")
        (mouse-set-point event)
        (org-open-at-point)))
    ;; Find all image overlays in the file and attach the custom click map
    (dolist (ov (overlays-in (point-min) (point-max)))
      (when (overlay-get ov 'org-image-overlay)
        (overlay-put ov 'keymap image-map)))))

;; Run this fix automatically every time Org mode draws images
(advice-add 'org-display-inline-images :after #'my-org-image-click-fix)

Auto open treemacs for coding projects
#+begin_src emacs-lisp
(defun my/auto-open-treemacs ()
  "Automatically open Treemacs if the current file belongs to a project."
  (when (and (not (treemacs-get-local-window)) ; Don't do anything if treemacs is already open
             (require 'project nil t)          ; Ensure project.el is loaded
             (project-current)                 ; Check if we are actually in a project
             (not (minibufferp)))              ; Don't trigger when navigating the minibuffer
    
    ;; Save selected window prevents Treemacs from stealing your cursor focus
    (save-selected-window
      ;; This function opens treemacs and isolates it to ONLY the current project
      ;; which fits the minimalist vibe perfectly.
      (treemacs-display-current-project-exclusively))))

;; Attach the function to the file-opening hook
(add-hook 'find-file-hook #'my/auto-open-treemacs)
